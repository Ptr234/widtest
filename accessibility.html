Accessibility Statement

Women in Design Uganda is committed to ensuring digital accessibility for people with disabilities.

Last Updated: December 1, 2025

1. Our Commitment to Accessibility

Women in Design Uganda is committed to making our website accessible to all individuals, regardless of ability. We believe that accessibility is not just a legal requirement, but a moral imperative. Our goal is to provide an inclusive digital experience for everyone who visits our site.

2. Standards and Guidelines

We have designed our website to comply with the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA, which is recognized as the international standard for web accessibility. Our website is designed to be:

  • Perceivable: Content is presented in ways that can be perceived by users with various disabilities
  • Operable: Website components are navigable using keyboard, voice commands, or other assistive technologies
  • Understandable: Text is clear and readable, with predictable navigation
  • Robust: Content is compatible with current and future assistive technologies

3. Accessibility Features

Our website includes the following accessibility features:

  • Semantic HTML: Proper use of HTML structure for screen readers
  • Keyboard Navigation: Full keyboard navigation support throughout the site
  • Skip Links: Skip to main content links on every page
  • Focus Indicators: Clear focus indicators for keyboard navigation
  • Alt Text: Descriptive alt text for all images
  • Color Contrast: Sufficient color contrast ratios for readability
  • Readable Fonts: Clear, sans-serif fonts with appropriate sizing
  • Form Labels: All form inputs have associated labels
  • Error Handling: Clear error messages and suggestions for correction
  • Mobile Responsive: Content is accessible on all device sizes

4. Assistive Technology Support

Our website is tested and compatible with the following assistive technologies:

  • Screen readers (NVDA, JAWS, VoiceOver)
  • Voice control software
  • Keyboard-only navigation
  • Zoom and magnification tools
  • High contrast mode

5. Known Accessibility Issues

While we strive for complete accessibility, we acknowledge that some third-party content or embedded media may have accessibility limitations. If you encounter any accessibility barriers, please let us know so we can work to address them.

6. Accessibility of Third-Party Content

Our website may contain links to external websites and embedded third-party content (such as videos or maps). We are not responsible for the accessibility of these external resources. However, we encourage all content providers to maintain accessible standards.

7. Browser and Device Compatibility

Our website has been tested for accessibility on the following browsers:

  • Google Chrome (latest version)
  • Mozilla Firefox (latest version)
  • Apple Safari (latest version)
  • Microsoft Edge (latest version)

8. Accessibility Adjustments

Many browsers and devices offer built-in accessibility features. Learn how to:

  • Increase Text Size: Use browser zoom or text size settings
  • Enable High Contrast: Use your operating system's high contrast settings
  • Use Screen Readers: Enable and configure your preferred screen reader
  • Keyboard Navigation: Use Tab key to navigate, Enter to select

9. Feedback and Improvements

We are continuously working to improve the accessibility of our website. If you encounter any accessibility barriers or have suggestions for improvement, we would love to hear from you.

10. Contact for Accessibility Issues

If you experience difficulty accessing any content on our website or would like to report an accessibility issue, please contact us:

11. Accessibility Certification

Our website has been evaluated for accessibility compliance. We are committed to maintaining and improving these standards. Our team regularly tests the website using both automated tools and manual testing with assistive technologies.

12. Resources for Users

For more information about web accessibility, visit:

  • Web Accessibility Initiative (WAI)
  • Web Content Accessibility Guidelines (WCAG)
  • Section 508 Compliance Information

Note: Accessibility is an ongoing commitment. We appreciate your feedback as it helps us create a more inclusive digital experience for everyone.
